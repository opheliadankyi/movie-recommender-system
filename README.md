# 🎬 Movie Recommender System

An AI-powered movie recommendation web application built with Flask, machine learning concepts, collaborative filtering, and TMDB API integration. The system recommends movies dynamically and displays real movie posters inside a modern interactive interface.

🌐 **Live Demo**  
https://movie-recommender-system-t2zj.onrender.com

---

# 🚀 Project Overview

This project was developed as part of an Applied Machine Learning and Recommendation Systems workflow. It combines recommendation system concepts with a fully deployed web application.

The application allows users to:
- Search movies by title or genre/category
- Receive intelligent movie recommendations
- Explore movie categories visually
- View real movie posters fetched dynamically from TMDB API
- Experience a modern responsive interface deployed online
