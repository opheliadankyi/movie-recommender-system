# 🎬 Movie Recommender System

An AI-powered movie recommendation web application built using Flask, collaborative filtering, and TMDB API integration.

## 🚀 Live Demo

https://movie-recommender-system-t2zj.onrender.com

---

## 📌 Features

- Movie recommendation system
- Dynamic movie poster fetching using TMDB API
- Search-based recommendations
- Category-based suggestions
- Responsive modern interface
- Flask backend deployment on Render
- Interactive UI design

---

## 🛠 Technologies Used

- Python
- Flask
- Pandas
- HTML/CSS
- TMDB API
- Render
- Collaborative Filtering

---

## 🎥 Example Categories

- Action Movies
- Fantasy Movies
- Romance Movies
- Science Fiction
- Adventure Movies
- Drama Movies
- Cartoon Movies

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/opheliadankyi/movie-recommender-system.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```
